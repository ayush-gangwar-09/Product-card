# 🛍️ Product Card

Internet Working Programming Laboratory - Mini Project

## 📌 About the Project

This project is a simple and responsive Product Card built using HTML and CSS.

It displays product details like image, name, price, and description in a visually appealing card layout.

This project was created as part of my learning in Internet Working Programming Laboratory to understand UI design and frontend basics.

Attractive product card UI

Responsive design

Clean layout

Easy to customize
